# Netflix Data Analysis

## Tools
Python, Pandas, Matplotlib

## Overview
Basic exploratory data analysis on Netflix dataset.

## Insights
- Movies significantly outnumber TV Shows on Netflix  
- The USA dominates Netflix content production by a wide margin  
- Content additions grew rapidly after 2015  
- Netflix content additions peaked around 2019 before slightly declining  

## Conclusion
This project analyzes Netflix content trends using Pandas and visualization.
